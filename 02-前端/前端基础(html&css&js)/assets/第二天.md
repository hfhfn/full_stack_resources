## 2. CSS外边距margin

+ margin的作用
+ margin的用法

#### 2.1 margin的作用

margin 属性在一个声明中设置所有外边距属性,该属性可以有 1 到 4 个值.

通俗的说来设置块元素的在网页的布局.

#### 2.2 margin的用法

+ 全写方式
  + margin-top
    + 上
  + margin-bottom
    + 下
  + margin-left
    + 左
  + margin-right
    + 右

+ 简写方式

  + 第一种

    ```html
    margin:10px 5px 15px 20px;
    
    上外边距是 10px
    右外边距是 5px
    下外边距是 15px
    左外边距是 20px
    ```

  + 第二种

    ```html
    margin:10px 5px 15px;
    
    上外边距是 10px
    右外边距和左外边距是 5px
    下外边距是 15px
    ```

  + 第三种

    ```html
    margin:10px 5px;
    
    上外边距和下外边距是 10px
    右外边距和左外边距是 5px
    ```

  + 第四种

    ```html
    margin:10px;
    
    所有 4 个外边距都是 10px
    ```

示例

```html
<style>
        p{
            width: 100px;
            height: 100px;
            background-color: red;
            /* 全写 */
            /* margin-left: -50px;
            margin-top: 100px;
            margin-bottom: 100px;
            margin-right: 50px; */
        }
        div{
            width: 200px;
            height: 200px;
            /* 简写 */
            background-color: green;
            /* 上 右 下 左 顺时针 */
            /* margin: 50px 100px 150px 200px; */
            /* 上 (左右) 下 */
            /* margin: 50px 100px 150px; */
            /* (上下) (左右) */
            /* margin: 50px 100px; */
            /* (上下左右) */
            /* margin: 50px; */
            /* 如果想水平居中 */
            margin:0px auto;
        }   
    </style>
</head>
<body>
    <p>
        段落标签
    </p>
    <div>
        div标签
    </div>   
</body>
```

>小结:
>
>允许使用负值.
>
>如果想完成水平居中 设置margin:0px auto;

## 4. CSS常用选择器一

学习目标

+ 选择器的作用
+ 标签选择器
+ 类选择器
+ 层级选择器(后代选择器)

#### 4.1 选择器的作用

实际开发中,我们可能需要完成以下需求:

需求1:

```html
<body>
    <!-- 要求:
        1. 宽度100px;
        2. 高度200px;
        3. 背景色红色;
     -->
    <div>
        div标签
    </div>
</body>
```

需求2:

```html
<body>
    <!-- 要求:
        1. 宽度100px;
        2. 高度200px;
        3. 背景色红色;
     -->
    <div>
        第一个div标签
    </div>
    <!-- 要求:
        1. 宽度200px;
        2. 高度400px;
        3. 背景色绿色;
     -->
    <div>
        第二个div标签
    </div>
</body>
```

需求3:

```html
<body>
   	<!-- 要求:
        设置p标签中文字的颜色为蓝色
     -->
    <div>
        div标签
        <p>段落标签</p>
    </div>
</body>
```



我们如何解决呢?我们可以利用选择器完成以上需求.

在 CSS 中，选择器是一种模式，用于选择需要添加样式的元素。

通俗的说,就是通过选择器找到对应的标签,然后对它进行装修.

#### 4.2 标签选择器

格式

```html
<style>
    标签名{
        属性1:属性值1;
        属性2:属性值2;
        ..:..;
	}
</style>
```

示例

```html
<style>
    /* <!-- 1. 标签选择器 --> */
    div{
        /* 1. 宽度100px; */
        width: 100px;
        /* 2. 高度200px; */
        height: 200px;
        /* 3. 背景色红色; */
        background-color: red;
    }
</style>
```

#### 4.3 类选择器

格式

```html
<style>
    .class的属性值{
        属性1:属性值1;
        属性2:属性值2;
        ..:..;
	}
</style>
```

首先需要分别给div设置一个class属性

```html
<body>
    <!-- 要求:
        1. 宽度100px;
        2. 高度200px;
        3. 背景色红色;
     -->
    <div class='first'>
        第一个div标签
    </div>
    <!-- 要求:
        1. 宽度200px;
        2. 高度400px;
        3. 背景色绿色;
     -->
    <div class='second'>
        第二个div标签
    </div>
</body>
```

示例

```html
<style>
    /* 类选择器 */
    .first{
        width: 100px;
        height: 200px;
        background-color: red;
    }

    .second{
        width: 200px;
        height: 400px;
        background-color: green;
    }

</style>
```

#### 4.4 层级选择器

也成为后代选择器

格式

```html
<style>
    标签名或者.class的属性值 标签名或者.class的属性值 {
        属性1:属性值1;
        属性2:属性值2;
        ..:..;
	}
</style>
```

首先需要分别给div和p设置一个class属性

```html
<body>
    <!-- 要求:
         设置p标签中文字的颜色为蓝色
      -->
    <div class='box'>
        div标签
        <p class='luo'>段落标签</p>
    </div>
</body>
```

示例

```html
<style>
    /* 层级选择器 */
    /*
    } */
    /* div .luo{
    color:blue;
    } */
    /* .box .luo{
    color:blue;
    } */

</style>
```

>小结:
>
>标签选择器:一般不用,杀伤力比较大,因为标签名一样会全部修改
>
>类选择: 有选择性的修改标签的属性
>
>层级选择器: 一般嵌套关系的时候使用

## 7. 元素溢出

学习目标

+ 什么是元素溢出
+ 元素溢出的解决方案

#### 7.1 什么是元素溢出

![元素溢出演示](assets/元素溢出演示.png)

当子元素的尺寸超过父元素的尺寸时,称为元素溢出.

#### 7.2 元素溢出的解决方案

需要设置父元素显示溢出的子元素的方式,设置的方法是通过overflow属性来设置.

overflow的设置项： 

+ visible      默认值。内容不会被修剪，会呈现在元素框之外。
+ hidden    内容会被修剪，并且其余内容是不可见的。
+ scroll       内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
+ auto        如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。  

>扩展:
>
>如果遇到长单词需要换行
>
>word-wrap:break-word;

## 10. 表单

+ 表单的作用
+ 表单的组成

#### 10.1 表单的作用

表单用于搜集不同类型的用户输入，表单由不同类型的标签组成

#### 10.2 表单的组成

表单自己

+ 标签名字
  + form

+ 标签属性

  + action
    + 定义表单数据提交地址

  + method
    + 定义表单提交的方式，一般有“get”方式和“post”方式

表单内部

+ label标签
  + 为表单元素定义文字标注

+ input标签

  + 定义通用的表单元素
  + type属性
    + type="text" 定义单行文本输入框
    + type="password" 定义密码输入框
    + type="radio" 定义单选框
    + type="checkbox" 定义复选框
    + type="file" 定义上传文件
    + type="submit" 定义提交按钮
    + type="reset" 定义重置按钮
    + type="button" 定义一个普通按钮

  + name属性 定义表单元素的名称，此名称是提交数据时的键名
  + value属性 定义表单元素的值

+ select标签
  + 定义下拉表单元素

+ option标签
  + 与select标签配合，定义下拉表单元素中的选项

+ textarea标签 
  + 定义多行文本输入框

示例

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>表单二</title>

    <style>
        .box{
            width: 500px;
            height: 600px;
            border: 1px red solid;
            margin: 0 auto;
        }
        .box h2{
            text-align: center;
        }
        
    </style>
</head>

<body>

    <!-- 表单 -->
    <form class='box'>
        <!-- 1.个人信息等级表 -->
        <h2>个人信息等级表</h2>
        <!-- 2. 用户名 -->
        <p>
            <label>用户名:</label>
            <input type="text" name="username">
        </p>
        <!-- 3.密码 -->
        <p>
            <label>密&nbsp;&nbsp;&nbsp;码:</label>
            <input type="password" name="pwd">
        </p>

        <!-- 4.性别 -->
        <p>
            <label>性&nbsp;&nbsp;&nbsp;别:</label>
```

## 14. 定位

学习目标

+ 相对定位
+ 绝对定位
+ 固定定位

定位主要完成设置块元素style属性中的position属性值

+ position
  + static  默认定位
  + relative 相对定位
  + absolute 绝对定位
  + fixed 固定定位

#### 14.1 相对定位

元素偏移某个距离,元素仍保持其未定位前的形状,它原本所占的空间仍保留.

+ 参照物
  + 自己

+ 是否占位置
  + 占

#### 14.2 绝对定位

元素原先在正常文档流中所占的空间会关闭(脱离文档流),就好像元素原来不存在一样.

+ 参照物
  + 默认浏览器
  + 也可以是父元素,但是父元素必须要设置定位

+ 是否占位置
  + 不占(因为脱离了文档流)

#### 14.3 固定定位

元素的表现类似于将 position 设置为 absolute(也会脱离文档流),但是元素的位置会固定.

+ 参照物
  + 浏览器

+ 是否占位置
  + 不占(因为脱离了文档流)

>小结
>
>文档流(文档流)
>文档流，是指盒子按照html标签编写的顺序依次从上到下，从左到右排列，块元素占一行，行内元素在一行之内从左到右排列，先写的先排列，后写的排在后面，每个盒子都占据自己的位置。
>
>
>
>z-index属性完成的是元素的层级,数值为整数,数值越大层级越高
>
>opacity属性设置透明度,数值范围(0~1)之间
>
>filter:alpha(opacity=30)适配低版本的ie浏览器

## 15. background属性

学习目标

+ background属性解释
+ background属性连写

#### 15.1 background属性解释

background属性是css中应用比较多，且比较重要的一个属性，它是负责给盒子设置背景图片和背景颜色的，background是一个复合属性，它可以分解成如下几个设置项：

+ background-image 设置背景图片地址
  + background-image: url('./images/bg.jpg');
+ background-repeat 设置背景图片如何重复平铺
  + background-repeat: no-repeat;
+ background-position 设置背景图片的位置
  + background-position: center;
  + background-position: 10px 20px;

background-color 设置背景颜色

- background-color :red;

#### 15.2 background属性连写

```html
background: url('./images/bg.jpg') no-repeat 10px 20px green;
书写不需要关心他的顺序, 会自动识别
```
