## 1. jQuery的选择器的过滤

学习目标

+ has
+ not
+ eq

已知html

```html
<body>
    <div class='box'>1</div>
    <div id='div1'>2</div>
    <div>
        3
        <p>段落</p>
    </div>
    <div>4</div>
    <div>5</div>
    <a href="#">链接标签</a>
</body>
```

#### 1.1 has

示例

```js
// 通过选择器获取标签
// 获取的其实是包含了5个div标签的集合
var $divs = $('div');
// 只选择$divs中包含p元素的div元素
var $div = $divs.has('p');
```

#### 1.2 not

示例

```js
// 通过选择器获取标签
// 获取的其实是包含了5个div标签的集合
var $divs = $('div');
// 只选择$divs中class不等于myClass的div元素
var $div = $divs.not('.box');
```

#### 1.3 eq

示例

```js
// 通过选择器获取标签
// 获取的其实是包含了5个div标签的集合
var $divs = $('div');
// 只选择$divs中下标为2的div元素
var $div = $divs.eq(2);
```

>小结:
>
>值得注意的是,我们在使用以上关键字的时候,都是判断的是$divs这个集合的标签,而和div以外的a标签没有任何关系.



## 2. jQuery选择器的转移

学习目标

+ jQuery选择器的转移的作用
+ jQuery选择器的转移的格式

#### 2.1 jQuery选择器的转移的作用

在页面中如果有大量的标签的时候,我们可以通过选择器的转移,减少给标签设置class或者id的属性,减少工作量,而且开发起来更加灵活.

#### 2.2 jQuery选择器的转移的格式

通过选择器的转移的特点可以分为两大类

兄弟关系

+ prev() 上一个标签
+ prevAll() 上全部标签
+ next() 下一个标签
+ nextAll() 下全部标签
+ siblings() 取反标签

```html
<div>
    父标签
    <div>儿子标签1</div>
    <div>儿子标签2</div>
    <div>儿子标签3</div>
    <div class='box'>儿子标签4</div>
    <div>儿子标签5</div>
    <div>儿子标签6</div>
    <div>儿子标签7</div>
</div>
```

示例

```js
$(function () {
    // 我们以儿子标签4位参照物
    var $div = $('div .box');
    // 上一个
    var $element = $div.prev();
    // 上全部
    $element = $div.prevAll();
    // 下一个
    $element = $div.next();
    // 下全部
    $element = $div.nextAll();
    // 除了自己以外
    $element = $div.siblings();
    // 设置文字颜色为红色
    $element.css({'color':'red'});  
});
```

父子关系

+ children() 所有的子标签
+ find() 某个子标签
+ parent()父标签

```html
<div>
    父标签
    <div>儿子标签1</div>
    <div>儿子标签2</div>
    <div>儿子标签3</div>
    <div class='box'>
        儿子标签4
        <h4>孙子标签1</h4>
        <p>孙子标签2</p>
    </div>
    <div>儿子标签5</div>
    <div>儿子标签6</div>
    <div>儿子标签7</div>
</div>
```

示例

```js
$(function () {
    // 我们以儿子标签4位参照物
    var $div = $('div .box');
    // 所有的子标签
    var $element = $div.children();
    // 某一个子标签
    $element = $div.find('h4');
    // 父标签
    $element = $div.parent();
	// 设置文字颜色为红色
    $element.css({'color':'red'});  
});
```

## 5. 点击事件this和index()

学习目标

+ this和index()的作用

#### 5.1 点击事件的设置方式

语法格式

```js

    // 监听点击做的事情
});
```

#### 5.2 this和index()的作用

+ $(this) 代表操作的对象
+ index()代表操作的对象的下标

示例

```html
<!DOCTYPE html>
<head>
    <meta charset="UTF-8">
    <title>点击事件this和index()</title>
    <script src="./js/jquery-1.12.4.min.js"></script>
    <style>
        .box{
            width: 300px;
            height: 300px;
            background: red;
        }
    </style>
    <script>
        $(function () {
            // 01
            // 获取标签
            var $div = $('.box');
            // 绑定点击事件
            $div.click(function () {
                // 点击设置为绿色
                // 关键 $(this) 代表操作的对象
                $(this).css({'background':'green'});
            });
            // 02
            // 获取标签
            var $li =  $('.list li');
            // 绑定点击事件
            $li.click(function () {
                // 点击设置为绿色
                // 关键 $(this) 代表操作的对象
                $(this).css({'color':'green'});
                // 获取对应的下标
                alert( $(this).index());
            });

        });
    </script>
</head>
<body>
    <!-- 01 -->
    <div class='box'></div>
    <!-- 02 -->
    <ul class="list">
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
    </ul>   
</body>
</html>
```

## 8. 选项卡案例

学习目标

+ 完成选项卡案例HTML

+ 完成选项卡案例CSS

+ 完成选项卡案例JS


#### 8.1 选项卡案例分析



![选项卡案例](assets/选项卡案例.jpg)

+ 最外层div (box)(600px * 400px 颜色随意)

+ 顶部div(up)(600px *100px 颜色随意)

  + ul (600px *100px 颜色随意)
    + li (200px *100px 红色) (字体 30px 白色)
    + li (200px *100px 绿色) (字体 默认 黑色)
    + li (200px *100px 蓝色) (字体 默认 黑色)

  + 底部div(bottom)
    + div (600px *300px 红色)(字体80px)
    + div (600px *300px 绿色)(字体80px)
    + div (600px *300px 蓝色)(字体80px)

#### 8.2 选项卡案例实现

示例

```html
<!DOCTYPE html>
<head>
    <meta charset="UTF-8">
    <title>-选项卡案例</title>
    <script src="./js/jquery-1.12.4.min.js"></script>
    <style>
        /* 外层 */
        .box{
            width: 600px;
            height: 400px;
            background-color: red;
            margin: 100px auto;
        }
        /* 顶部 */
        .up{
            height: 100px;
            background-color: antiquewhite;
        }
        /* 顶部ul */
        .up ul{
            list-style: none;
            padding: 0px;
        }
        /* 顶部ul li */
        .up ul li{
            width: 200px;
            height: 100px;
            float: left;
            line-height: 100px;
            text-align: center;
        }

        /* 底部div */
        .down div {
            width: 600px;
            height: 300px;
            font-size: 80px;
            line-height: 300px;
            text-align: center;
            /* 隐藏 */
            display: none;
        }
        /* li样式 */
        .box .up .action{
            color: white;
            font-size: 30px;
        }

        /* div样式 */
        .box .down .current{
            color: white;
            display: block;
        }
        
    </style>

    <script>
        $(function () {
            // 获取三个按钮
            var $btns = $('.up ul li');
            // 获取三个div
            var $divs = $('.down div');
            // 事件
            $btns.click(function () {
                // btn
                $(this).addClass('action');
                $(this).siblings().removeClass('action');
                // div
                $divs.eq($(this).index()).addClass('current');
                $divs.eq($(this).index()).siblings().removeClass('current');
            });
        });

    </script>
</head>
<body>
    <!-- 外部div -->
    <div class='box'>
        <!-- 顶部 -->
        <div class="up">
            <ul>
                <li class='action' style="background-color:red;">红色按钮</li>
                <li style="background-color:green;">绿色按钮</li>
                <li style="background-color:blue;">蓝色按钮</li>
            </ul>
        </div>
        <!-- 底部 -->
        <div class="down">
            <div class='current' style="background-color:red;">红色div</div>
            <div style="background-color:green;">绿色div</div>
            <div style="background-color:blue;">蓝色div</div>
        </div>
    </div>
    
</body>
</html>
```



## 9.  jQuery动画

学习目标

+ jQuery动画语法格式

#### 9.1  jQuery动画语法格式

```js
$('选择器').animate({params},speed,easing,callback);
```

+ params 代表的是标签对象的style属性
+ speed 动画时长, 单位毫秒
+ easing 动画样式
  + swing (先慢后快)(默认)
  + linear(均速)

+ callback 动画完成后执行的代码(匿名函数)

示例

```js
$('#div1').animate(
    {
        'width':'1000'
    },
    2000,
    'linear',
    function () {
        console.log('动画完成了.');  
    }
);
```

>小结:
>
>默认动画只能设置的为px结尾的属性.
>
>可能会修改颜色,那么需要我们导入.
>
>```js
><script src="./js/jquery.color.js"></script>
>```
>
>它是动画的扩展文件.
