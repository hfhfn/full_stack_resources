## 1. 事件属性及匿名函数

学习目标

+ 匿名函数配合事件属性使用

#### 1.1 匿名函数配合事件属性使用

元素上除了有样式，id等属性外，还有事件属性，常用的事件属性有鼠标点击事件属性(onclick)，鼠标移入事件属性(mouseover),鼠标移出事件属性(mouseout),将函数名称赋值给元素事件属性，可以将事件和函数关联起来。

```html
<!DOCTYPE html>
<head>
    <meta charset="UTF-8">
    <title>事件属性及匿名函数</title>
    <script>
        window.onload = function () {
            // 定义一个函数完成弹出提示框
            function fnAlert() {
                alert('我来了');
            }
            // 获取input标签
            var oBtn = document.getElementById('btn');

            // 监听按钮点击
            oBtn.onclick = fnAlert;
        }
    </script>
</head>
<body>
    <input type="button" value="弹出提示框" id='btn'>
</body>
</html>
```

定义的函数可以不给名称，这个叫做匿名函数，可以将匿名函数的定义直接赋值给元素的事件属性来完成事件和函数的关联，这样可以减少函数命名，并且简化代码。函数如果做公共函数，就可以写成匿名函数的形式。

```html
<script>
    window.onload = function () {
        // 获取input标签
        var oBtn = document.getElementById('btn');

        // 监听按钮点击
        oBtn.onclick = function (){
            alert('我来了');
        }
    }
</script>
```

#### 4.3 数组的操作

+ **push**

  + 向数组的尾部追加一个元素

  ```js
  数组名.push(数值);
  ```

  示例

  ```js
  var aOne = [1, 2, 3];
  aOne.push(4);
  // 结果
  [1, 2, 3, 4]
  ```

+ **pop**

  + 删除最后一个元素

  ```js
  数组名.pop();
  ```

  示例

  ```js
  var aOne = [1, 2, 3];
  aOne.pop();
  // 结果
  [1, 2]
  ```

+ **lenght**

  + 计算数组长度

  ```js
  var len = 数组名.lenght;
  ```

  示例

  ```js
  var aOne = [1, 2, 3];
  var len = aOne.length;
  // 结果
  3
  ```

+ **indexOf**

  + 查看元素的下标
    + 元素存在返回下标
    + 元素不存在返回-1

  ```js
  var index = 数组名.indexOf(元素);
  ```

  示例

  ```js
  var aOne = [1, 2, 3];
  // 存在
  var index = aOne.indexOf(2);
  // 结果
  1
  -----------------------------
  // 不存在
  var index = aOne.indexOf(22);
  // 结果
  -1
  ```

+ **reverse**

  + 完成数组元素倒序

  ```js
  数组名.reverse()
  ```

  示例

  ```js
  var aOne = [1, 2, 3]; 
  aOne.reverse();
  // 结果
  [3, 2, 1]
  ```

+ **join**

  + 数组类型转字符串类型

  ```js
  数组名.join();
  // 或者
  数组名.join(字符串);
  ```

  示例1:

  ```js
  var aOne = [1, 2, 3]; 
  var result = aOne.join();
  // 结果
  '1,2,3'
  ```

  示例2:

  ```js
  var aOne = [1, 2, 3]; 
  var result = aOne.join('-');
  // 结果
  '1-2-3'
  ```

+ **splice**

  + 数组中增加或删除元素

  ```
  数组名.splice(index,howmany,item1,.....,itemX);
  ```

  | 参数              | 描述                                                  |
  | ----------------- | ----------------------------------------------------- |
  | index             | 必需. 整数,规定添加/删除项目的位置.                   |
  | howmany           | 必需. 要删除的元素数量, 如果设置为 0, 则不会删除元素. |
  | item1, ..., itemX | 可选. 向数组添加的新项目.                             |

## 8. 封闭函数(了解)

学习目标

+ 封闭函数的格式
+ 封闭函数的作用

#### 8.1 封闭函数的格式

第一种格式:

```js
(function () {
    alert('封闭函数');
})();
```

第二种:

```js
~function () {
    alert('封闭函数'); 
}();
```

第三种:

```js
!function () {
    alert('封闭函数'); 
}();
```



#### 8.2 封闭函数的作用

+ 函数只会执行一次
  + 想把多行代码保存到一个函数中,便于维护管理
  + 程序一启动就想执行
+ 避免函数名冲突
  + 工作后接收老项目
  + 有时候出现函数名冲突的解决方案

示例:

```js
function fnAlert() {
    alert('老家伙');
}

fnAlert();

function fnAlert() {
    alert('帅小伙');
}

fnAlert();
```

浏览器会弹出两次提示框,而且每次提示内容都是帅小伙

不要忘记了函数的预解析

解决方案:

```js
function fnAlert() {
    alert('老家伙');
}

fnAlert();

~function () {
    function fnAlert() {
        alert('帅小伙');
    }
    fnAlert();

}();
```

>小结:
>
>封闭函数可以创造一个独立的空间, 在封闭函数内定义的变量和函数不会影响外部同名的函数和变量, 可以避免命名冲突, 在页面上引入多个js文件时, 用这种方式添加js文件比较安全.

## 13. jQuery选择器的转移

学习目标

+ jQuery选择器的转移的作用
+ jQuery选择器的转移的格式

#### 13.1 jQuery选择器的转移的作用

在页面中如果有大量的标签的时候,我们可以通过选择器的转移,建设给标签设置class或者id的属性,减少工作量,而且开发起来更加灵活.

#### 13.2 jQuery选择器的转移的格式

通过选择器的转移的特点可以分为两大类

兄弟关系

+ prev() 上一个标签
+ prevAll() 上全部标签
+ next() 下一个标签
+ nextAll() 下全部标签
+ siblings() 取反标签

```html
<div>
    父标签
    <div>儿子标签1</div>
    <div>儿子标签2</div>
    <div>儿子标签3</div>
    <div class='box'>儿子标签4</div>
    <div>儿子标签5</div>
    <div>儿子标签6</div>
    <div>儿子标签7</div>
</div>
```

示例

```js
$(function () {
    // 我们以儿子标签4位参照物
    var $div = $('div .box');
    // 上一个
    var $element = $div.prev();
    // 上全部
    $element = $div.prevAll();
    // 下一个
    $element = $div.next();
    // 下全部
    $element = $div.nextAll();
    // 除了自己以外
    $element = $div.siblings();
    // 设置文字颜色为红色
    $element.css({'color':'red'});  
});
```

父子关系

+ children() 所有的子标签
+ find() 某个子标签
+ parent()父标签

```html
<div>
    父标签
    <div>儿子标签1</div>
    <div>儿子标签2</div>
    <div>儿子标签3</div>
    <div class='box'>
        儿子标签4
        <h4>孙子标签1</h4>
        <p>孙子标签2</p>
    </div>
    <div>儿子标签5</div>
    <div>儿子标签6</div>
    <div>儿子标签7</div>
</div>
```

示例

```js
$(function () {
    // 我们以儿子标签4位参照物
    var $div = $('div .box');
    // 所有的子标签
    var $element = $div.children();
    // 某一个子标签
    $element = $div.find('h4');
    // 父标签
    $element = $div.parent();
	// 设置文字颜色为红色
    $element.css({'color':'red'});  
});
```

#### 16.2 this和index()的作用

```html

    <meta charset="UTF-8">

        }

            // 01
            // 获取标签
            var $div = $('.box');
            // 绑定点击事件
            $div.click(function () {
                // 点击设置为绿色

            // 02

                // 点击设置为绿色
                // 关键 $(this) 代表操作的对象

                // 获取对应的下标

            });

    </script>
```
