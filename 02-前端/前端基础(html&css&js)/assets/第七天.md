#### 1.2 AJAX的介绍

AJAX是Asynchronous JavaScript and XML的简写, （异步的JavaScript 和 XML ）.

AJAX 不是新的编程语言, 而是一种使用现有标准的新方法.

AJAX 最大的优点是在不重新加载**整个页面**的情况下, 可以与服务器交换数据并更新**部分网页内容**(局部刷新).

通俗的说,如果前端想和后台服务器进行数据交互就可以使用AJAX完成

## 3. AJAX基本写法

学习目标

+  AJAX基本写法

#### 3.1 AJAX基本写法

常用参数：
1.url 请求地址
2.type 请求方式，默认是'GET'，常用的还有'POST'
3.dataType 设置返回的数据格式，常用的是'json'格式，也可以设置为'html'
4.data 设置发送给服务器的数据
5.success 设置请求成功后的回调函数
6.error 设置请求失败后的回调函数
7.async 设置是否异步，默认值是'true'，表示异步

示例

```js
<script>
    $.ajax({
    // 1.url 请求地址
    url:'http://t.weather.sojson.com/api/weather/city/101010100',
    // url:'192.168.0.109',
    // 2.type 请求方式，默认是'GET'，常用的还有'POST'
    type:'GET',
    // 3.dataType 设置返回的数据格式，常用的是'json'格式，也可以设置为'html'
    dataType:'JSON',
    // 4.data 设置发送给服务器的数据

    // 5.success 设置请求成功后的回调函数
    success:function (response) {
        console.log(response);    
    },
    // 6.error 设置请求失败后的回调函数
    error:function () {
        alert("请求失败,请稍后再试!");
    },
    // 7.async 设置是否异步，默认值是'true'，表示异步
    async:true
});
</script>
```

#### 4.3 天气预报案例AJAX

示例

```js
<script>
        $(function () {
            // 获取 body左中右div
            var $body = $('.bodys'),
                $leftDiv = $('.left'),
                $centerDiv = $('.center'),
                $rightDiv = $('.right');
            // 颜色数组
            var aColors = ['red', 'green', 'blue'];
            // 更改颜色动画
            function fnChangeColor() {
                // 随机数的生成[0,3)
                // Math.floor(Math.random()*aColors.length)
                $body.animate(
                {
                    'background-color':aColors[Math.floor(Math.random()*aColors.length)]
                },
                1000,
                'linear',
                function () {
                    // 递归调用
                    fnChangeColor();
                }
                );
            }
            fnChangeColor();
            
            // 请求数据
            function fnLoadData(cityId) {
                // 发送请求
                $.ajax({
                    url:'http://t.weather.sojson.com/api/weather/city/' + cityId,
                    type:'get',
                    dataType:'json'
                })
                // 成功
                .done(function (response) {
                    console.log(response);
                    // 城市
                    $leftDiv.html(response.cityInfo.parent);
                    // 信息
                    var sContent = ''
                    + '<p>日期: ' + response.date + '</p>'
                    + '<p>温度: ' + response.data.wendu + '℃</p>'
                    + '<p>湿度: ' + response.data.shidu + '</p>'
                    + '<p>空气: ' + response.data.quality + '</p>'
                    + '<p>PM10: ' + response.data.pm10  + '</p>'
                    + '<p>PM25: ' + response.data.pm25+ '</p>'
                    $centerDiv.html(sContent);
                })
                // 失败
                .fail(function () {
                    alert('加载失败,请重试!')
                });
            }
            // 城市id数组
            var aCity = [101010100, 101020100, 101030100, 101040100];
            // 调用
            fnLoadData(aCity[0]);
		
            // 监听div点击
            $rightDiv.delegate('div', 'click', function () {
                // 局部刷新
                // 发送请求
                fnLoadData(aCity[$(this).index()]);
                // 添加样式类
                $(this).addClass('current');
                $(this).siblings().removeClass('current');

            });

        });
</script>
```

```html
        <!-- <p v-else-if='flag==3'>03:v-else-if</p> -->
        <!-- 04:v-else -->
        <!-- <p v-else>04:v-else</p> -->
        <!-- 05:v-show -->
        <!-- <p v-show='flag==3'>05:v-show</p> -->
    </div>
</body>
</html>
```

>小结:
>
>1. v-show用法和v-if大致一样，但是它不支持v-else,它和v-if的区别是，它制作元素样式的显示和隐藏，元素一直是存在的
>
>2. 注意在vue中使用v-show, 原来的css代码不能设置display属性, 会导致冲突

## 12. 列表渲染

学习目标

+ 普通列表
+ 列表下标
+ 有且只有一个对象
+ 对象列表

js部分

```js
<script>
    window.onload = function () {
    var vm = new Vue({
        el:'.box',
        data:{
            // 01: 普通列表
            itemList:[1, 2, 3, 4, 5],
            // 02: 列表下标
            indexList:['a','b','c','d'],
            // 03: 有且只有一个对象
            objData:{
                name:'小明',
                age:19
            },
            // 04: 对象列表
            objList:[
                {
                    name:'小明',
                    age:20
                },
                {
                    name:'小红',
                    age:21
                }
            ]
        }
    });
}
</script>  
```



#### 12.1 普通列表

```html
<li v-for='item in itemList'>{{item}}</li>
```

#### 12.2 列表下标

```html
<li v-for='(item,index) in indexList'>角标{{index}}==数值{{item}}</li>
```

#### 12.3 有且只有一个对象

```html
<li v-for='item in objData'>{{item}}</li>
<li v-for='(obj,key) in objData'>属性值{{obj}}-----属性名{{key}}</li>
```

#### 12.4 对象列表

```html
<li v-for='obj in objList'>属性值1:{{obj.name}}==属性值2:{{obj.age}}</li>
```

>小结:
>
>无非名字叫做列表渲染而已, 这种指令适合任何html标签
