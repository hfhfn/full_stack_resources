## 4. 正则表达式

学习目标

+ 回顾正则
+ js的正则写法

#### 4.1 回顾正则

+ 什么是正则
  + 能让计算机读懂的字符串匹配规则。

+ 规则中的字符 

  + 普通字符匹配
    + 是什么就是什么

  + 转义字符匹配
    + \d 匹配一个数字，即0-9
    + \D 匹配一个非数字，即除了0-9
    + \w 匹配一个单词字符（字母、数字、下划线）
    + \W 匹配任何非单词字符。等价于\[^A-Za-z0-9_]
    + \s 匹配一个空白符
    + \S 匹配一个非空白符
    + \b 匹配单词边界 '
    + \B 匹配非单词边界
    + . 匹配一个任意字符

  + 量词(对左边的匹配字符定义个数 )

    + ? 出现零次或一次（最多出现一次）
    + \+ 出现一次或多次（至少出现一次）
    + \* 出现零次或多次（任意次）

    + {n} 出现n次

    +  {n,m} 出现n到m次

    + {n,} 至少出现n次

  + 任意一个或者范围 
    + [abc123] : 匹配‘abc123’中的任意一个字符
    + [a-z0-9] : 匹配a到z或者0到9中的任意一个字符

  + 限制开头结尾 
    + ^ 以紧挨的元素开头
    + $ 以紧挨的元素结尾

  + 正则默认规则 
    + 匹配成功就结束，不会继续匹配，区分大小写

**常用正则规则**

```js
//用户名验证：(数字字母或下划线6到20位)
var reUser = /^\w{6,20}$/;

//邮箱验证：        
var reMail = /^[a-z0-9][\w\.\-]*@[a-z0-9\-]+(\.[a-z]{2,5}){1,2}$/i;

//密码验证：
var rePass = /^[\w!@#$%^&*]{6,20}$/;

//手机号码验证：
var rePhone = /^1[34578]\d{9}$/;
```

#### 4.2  js的正则写法

正则表达式的写法

```js
var re = new RegExp('规则', '可选参数');
var re = /规则/参数;
```

修饰参数

- g： global，全文搜索，默认搜索到第一个结果接停止
- i： ignore case，忽略大小写，默认大小写敏感

常用函数 

+ search 

  + 用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串，并返回子串的起始位置。

  ```js
  var str = "Hello World!"; 
  var n = str.search(/world/i);
  // 结果:
  6
  // 如果没有匹配到
  -1
  ```

+ replace

  + 用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。

  ```js
  var str = "Hello World!World!World!World!"; 
  var n = str.replace(/world/ig, 'Python');
  // 结果:
  Hello Python!Python!Python!Python!
  ```

+ test

  + 用于检测一个字符串是否匹配某个模式，如果字符串中含有匹配的文本，则返回 true，否则返回 false。

  ```js
  var str = "Hello World!"; 
  var re = /e/;
  var n = re.test(str);
  // 结果:
  true
  // 如果没有
  false
  ```

## 7. 事件委托

学习目标

+ 事件委托的作用

#### 7.1 事件委托的作用

事件委托就是利用冒泡的原理, 把事件加到父级上, 通过判断事件来源的子集, 执行相应的操作.

优点:

+ 可以极大减少事件绑定次数, 提高性能
+ 可以让新加入的子元素也可以拥有相同的操作

语法:

```js
 $被委托的标签名.delegate(委托的标签名, 事件名, 匿名函数);
```

**一般绑定事件的写法**

```js
$(function(){
    $ali = $('#list li');
    $ali.click(function() {
        $(this).css({background:'red'});
    });
    // 如果新增元素
    $list.html('<li>6</li>'); 
});
...
<ul id="list">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>
```

**事件委托的写法**

```js
$(function(){
    $list = $('#list');
    $list.delegate('li', 'click', function() {
        $(this).css({background:'red'});
    });
    // 如果新增元素
    $list.html('<li>6</li>');
});
...
<ul id="list">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>
```

>小结:
>
>在实际开发中, 如果有新增元素的需求, 利用事件委托才可以把问题解决掉

```js
$(function () {
```

```js
        // 监听

            // 输入框
            var $input01 = $('#input01');
            // 按钮
            var $input02 = $('#input02');
            // 列表
            var $list = $('.list');

                // 添加
                $list.append($content);
                // 清空数据
                $input01.val('');
               
            });

            // 事件委托
            // 注意是a被点击
            $list.delegate('a', 'click', function () {
                // 操作对象的class
                var sClass = $(this).prop('class');
                //  获取他的父元素
                var $parent =  $(this).parent();

                if (sClass=='delete') {
                    // 删除
                    $parent.remove();

                } else if (sClass=='up') {
                    // 上
                    // 上一个元素
                    var $up = $parent.prev();
                    // 判断是否有标签
                    if ($up.length == 0) {
                        alert('到头了');
                        return;
                    }
                    // $parent插入在$up前面
                    // $parent.insertBefore($up);
                    // $up的前面是$parent
                    $up.before($parent);
                    
                } else if (sClass=='down') {
                    // 下
                    // 下一个元素
                    var $down = $parent.next();
                    // 判断是否有标签
                    if ($down.length == 0) {
                        alert('到底了');
                        return;
                    }
                    // $parent插入在$up后面
                    // $parent.insertAfter($down);
                    //  $down的后面是$parent
                    $down.after($parent);
                    
                }
                // 上
                // 下
            });

        });

    </script>
```

#### 10.2 AJAX的介绍

AJAX是Asynchronous JavaScript and XML的简写, （异步的JavaScript 和 XML ）.

AJAX 不是新的编程语言, 而是一种使用现有标准的新方法.

AJAX 最大的优点是在不重新加载**整个页面**的情况下, 可以与服务器交换数据并更新**部分网页内容**(局部刷新).

通俗的说,如果前端想和后台服务器进行数据交互就可以使用AJAX完成

#### 13.3 天气预报案例AJAX

示例

```js
<script>
        $(function () {
            // 获取 body左中右div
            var $body = $('.bodys'),
                $leftDiv = $('.left'),
                $centerDiv = $('.center'),
                $rightDiv = $('.right');
            // 颜色数组
            var aColors = ['red', 'green', 'blue'];
            // 更改颜色动画
            function fnChangeColor() {
                // 随机数的生成[0,3)
                // Math.floor(Math.random()*aColors.length)
                $body.animate(
                {
                    'background-color':aColors[Math.floor(Math.random()*aColors.length)]
                },
                1000,
                'linear',
                function () {
                    // 递归调用
                    fnChangeColor();
                }
                );
            }
            fnChangeColor();
            
            // 请求数据
            function fnLoadData(cityId) {
                // 发送请求
                $.ajax({
                    url:'http://t.weather.sojson.com/api/weather/city/' + cityId,
                    type:'get',
                    dataType:'json'
                })
                // 成功
                .done(function (response) {
                    console.log(response);
                    // 城市
                    $leftDiv.html(response.cityInfo.parent);
                    // 信息
                    var sContent = ''
                    + '<p>日期: ' + response.date + '</p>'
                    + '<p>温度: ' + response.data.wendu + '℃</p>'
                    + '<p>湿度: ' + response.data.shidu + '</p>'
                    + '<p>空气: ' + response.data.quality + '</p>'
                    + '<p>PM10: ' + response.data.pm10  + '</p>'
                    + '<p>PM25: ' + response.data.pm25+ '</p>'
                    $centerDiv.html(sContent);
                })
                // 失败
                .fail(function () {
                    alert('加载失败,请重试!')
                });
            }
            // 城市id数组
            var aCity = [101010100, 101020100, 101030100, 101040100];
            // 调用
            fnLoadData(aCity[0]);
		
            // 监听div点击
            $rightDiv.delegate('div', 'click', function () {
                // 局部刷新
                // 发送请求
                fnLoadData(aCity[$(this).index()]);
                // 添加样式类
                $(this).addClass('current');
                $(this).siblings().removeClass('current');

            });

        });
</script>
```
