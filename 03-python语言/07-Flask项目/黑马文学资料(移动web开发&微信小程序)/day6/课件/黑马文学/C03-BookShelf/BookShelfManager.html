3.2书架管理

  • 在applet_app/mybooks.py文件中实现业务。
  • 书架的业务，主要是添加和删除书籍。

1-1 添加书籍的基本业务

  • 添加登录验证装饰器。
  • 需要传入参数书籍id
  • 查询书籍表，确认书籍的存在。
  • 查询书架表，如果书架上没有该书籍，添加书籍。
  • 否则，返回该书籍已在书架中。

1-3 删除书籍的基本业务

  • 根据传入的参数书籍id，查询书架表
  • 判断查询结果，如果书籍不存在，返回信息
  • 否则，删除书架中的该书籍数据

1、具体见代码：

        return jsonify(msg='该书籍在书架中不存在'),400
    # - 4.删除书籍
