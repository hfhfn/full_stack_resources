### 1.我的浏览记录

* 接口设计：

  * 请求地址：'/my/histories'
  * 请求方法：GET
  * 请求参数：

  ~~~python
  参数名				是否必须				参数类型				参数说明
  page					True					int					 查询字符串，页数
  pagesize			True					int					 查询字符串，每页的条目数
  
  ~~~

  * 返回数据：分页后的浏览数据，书籍名称、作者、封面图片、日期

* 实现步骤：
  * 1.新建my.py文件，用来实现关于浏览记录的功能代码；
  * 2.创建蓝图、定义蓝图、注册蓝图
  * 3.导入登录验证装饰器
  * 4.获取参数，page和pagesize
  * 5.查询数据库浏览记录表，根据用户id查询，分页处理
  * 6.获取分页后的数据
  * 7.转成json，返回数据
