### 3.数据库迁移：

* 前提：数据库必须先创建。
* 迁移：创建数据库表；需要使用两个扩展包；
  * flask-script：脚本管理器
  * flask-migrate：迁移的框架和命令

* 使用步骤：
  * 1、生成迁移仓库(文件夹)：python manage.py db init
  * 2、生成迁移脚本：python manage.py db migrate -m init_tables
  * 3、执行迁移脚本：python manage.py db upgrade，执行完成，数据库表创建成功。
