##### 1, 登录装饰器

- 目的: 可以编写登录装饰器, 判断用户的登录状态

- 操作流程:

  - 1, 登录装饰器(common/utils/login_required.py)

    - ```python
      from flask import request,current_app,g
      from itsdangerous import TimedJSONWebSignatureSerializer as Serializer,SignatureExpired,BadSignature
      
      #1,登录装饰器
      def login_required(view_func):
          def wrapper(*args,**kwargs):
      
              #1,获取到token
              token = request.headers.get("Authorization")
              print(token)
      
              #2,判断token正确性
              payload = None
              if token and token.startswith("Bearer "):
                  serializer = Serializer(secret_key=current_app.config.get("SECRET_KEY"),
                                          expires_in=current_app.config.get("TOKEN_HOURS"))
      
                  try:
                      payload = serializer.loads(token[7:])
                  except SignatureExpired:
                      payload = None
                  except BadSignature:
                      payload = None
      
              #3,设置g.user_id
              g.user_id = None
              if payload:
                  g.user_id = payload.get("user_id")
      
              #4,判断用户登录状态
              if g.user_id:
                  return view_func(*args,**kwargs)
              else:
                  return {"message":"用户未登录"},401
      
          return wrapper
      ```

  - 2, 类视图使用(lawyer/resources/question/question.py)

    - ```python

      def login_required(view_func):
          def wrapper(*args,**kwargs):
      
              #4,判断用户登录状态
              if g.user_id:
      
                  #4,1 获取前端携带的token
                  token = request.headers.get("Authorization")[7:]
      
                  #4,2 获取redis中的token
                  key = "token:{}".format(g.user_id)
                  token2 = current_app.redis_master.get(key)
      
                  #4,3 判断两个token是否一致, 如果不一致说明在多个设备登录
                  if token2 and token2.decode() != token:
                      return {"message":"用户已经在其他设备登录了"},403
      
                  ...
      ```
