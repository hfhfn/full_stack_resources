##### 2, 封装Queries查询器

- 目的: 封装Queries查询器, 简化查询方式和with

- 操作流程:

  - 1, 定义Queries查询器(common/utils/slave_db_router.py)

    - ```python
      #2,封装Queries查询器,对with进行简化
      class Queries(object):
          @classmethod
          def fetchone(self,sql,lst):
              with DBOpenExe(sql,lst) as cs:
                  data = cs.fetchone()
                  return data
      
          @classmethod
          def fetchall(self,sql,lst):
              with DBOpenExe(sql,lst) as cs:
                  data = cs.fetchall()
                  return data
      ```

  - 2, 测试结果(lawyer/main.py)

##### 9,jwt概念

- 目的: jwt作用和session区别即可
- jwt作用:
  - 1, 进行用户认证的字符串
  - 2, 服务器进行生成和校验即可, 具体的存在在浏览器
- session区别:
  - 1, session: 认证状态保持信息存储在服务器,不利于扩展
  - 2, jwt: 服务器生成, 存储在浏览器, 减轻了服务器的压力

      from itsdangerous import TimedJSONWebSignatureSerializer as Serializer
      from flask import current_app
      
      def generate_token(user_id,role,expires_in):
          #1,生成加密对象
          serializer = Serializer(secret_key=current_app.config.get("SECRET_KEY"), expires_in=300)
      
          #2,加密用户信息
          token = serializer.dumps({
              "user_id": user_id,
              "role": role
          }).decode()
      
          #3,返回token
          return token
