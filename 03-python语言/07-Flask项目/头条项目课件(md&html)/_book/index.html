课程简介

1. 课程内容

  • 项目介绍
  • 数据库方案
  • Gitflow工作流
  • 认证方案
  • 对象存储
  • 缓存方案
  • APScheduler定时任务
  • RPC方案 grpc
  • 即时通讯方案 socketio
  • 搜索方案 elasticsearch
  • 单元测试 unittest
  • supervisor进程管理

2. 目标

  • 以黑马头条产品为依托案例，但不再以具体业务实现为主
  • 力求掌握解决不同问题的实施方案
  • 深入理解并胜任python web开发
  • 具备一定的架构能力

3. 要遵循

  • 思考的作用大于动手练！但不动手写代码，思考就是白扯！
  • 不要死记硬背；只要掌握关键词，对大致流程有印象就好！
  • 先有广度，才能有深度！
  • 找工作面试的时候，以上三条就是屁话

4. 课程前置需要学习并了解

  • python语法
  • mysql数据库使用
  • redis的使用
  • Git的使用
  • django框架
  • flask框架
  • linux操作系统基本使用
  • TCP/IP以及HTTP协议

5. 其他

  • 建议内存16G，最低要求8G
  • 课程中使用了python3.6
  • 课程中IDE使用了pycharm
