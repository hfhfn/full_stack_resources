# 即时通讯

> 即时通信是基于`TCP长连接`, 建立连接之后, 客户端/服务器可以无限次随时向对端发送数据, 实现服务器数据发送的即时性
>
> 即时通信最常见的应用场景就是推送
>
> - 在线推送
>   - 适用：web页面 和 App
>   - 自己构建IM即时通讯服务
>     - 使用WebSocket
>     - 采用成熟的框架方案Socket.IO
>     - 对于App还可自己封装socket
>   - 使用第三方IM服务商提供的服务
>     - 网易云信
>     - 融云
>     - 环信
>     - LeanCloud
> - 离线推送
>   - 适用：App
>   - 对于iOS，使用APNs
>   - 对于andorid，使用FCM（国内用不了）或第三方IM服务商提供的服务
>     - 思考：某些安卓上app，没有使用第三方即时通讯服务，也能实现离线推送，为什么？
>
> **课程只针对 「在线推送」 的自建方案进行展开。**
